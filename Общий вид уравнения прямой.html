Ax + By + C = 0

Это общий вид уравнения прямой. Геометрический смысл A и B таков: это координаты вектора нормали к прямой (вектора, перпендикулярного задающейся прямой) (т.е. вектор нормали имеет координаты (A; B)), а C - это переменная, связанная с расстоянием до прямой от начала координат следующим уравнением: \(p = {C \over \sqrt{A^2 + B^2}}\). p и есть это расстояние. Общий вид уравнения прямой позволяет записать уравнение любой прямой на плоскости, в отличии от стандартного вида уравнения линейной функции, с помощью которого нельзя записать уравнение вертикальной прямой, но если записать уравнение прямой в общем виде с обнуленным y (задать B = 0), то графиком этого уравнения будет прямая, параллельная оси y, т.е. вертикальная прямая.

Также с помощью общего вида уравнения прямой можно записать уравнение любой линейной функции. Докажем это.

Возьмём стандартный вид уравнения линейной функции y = kx + b и перенесём kx и b налево, получим: y - kx - b = 0. Перегруппируем следующим образом: -kx + y - b = 0. Ничего не напоминает? Это уравнение прямой в общем виде, где A = -k, B = 1, а C = -b, что и требовалось доказать.
