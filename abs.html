y = ax2 + bx + c

Это стандартный вид квадратичной функции. Из-за того, что зависимость значения функции от значения аргумента не линейная, а квадратичная, то графиком этой функции является кривая, называемая параболой. Парабола состоит из двух ветвей. Старший коэффициент a определяет направление ветвей параболы: если a > 0, то ветви параболы направлены вверх (изображено на рисунке); если a < 0, то ветви параболы направлены вниз. Коэффициент b влияет на расположение вершины параболы (точки, где две ветви соединяются): вершина имеет абсциссу

x0

-b 2a

Коэффициент c поднимает или опускает график на c делений по оси ординат: если c > 0, то график будет поднят; если c < 0, то график будет опущен. Также коэффициент c равен ординате точки пересечения параболы с осью y.

Абсциссы точек пересечения параболы с осью x равны корням уравнения ax2 + bx + c = 0. По значению дискриминанта D = b2 - 4ac можно определить количество этих точек: при D < 0 таких точек не будет, при D = 0 будет одна такая точка, а при D > 0 будет две такие точки.
